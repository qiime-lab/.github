# qiime-lab

**QIIME 2 マイクロバイオーム解析のためのオープンソースツール群**

16S rRNA アンプリコンシーケンス解析を、より簡単に・より再現性高く・より自動的に。

---

## Projects

| Repository | Description |
|:--|:--|
| **[seq2pipe](https://github.com/qiime-lab/seq2pipe)** | ローカル LLM エージェントによる QIIME 2 解析の完全自動化パイプライン。FASTQ → DADA2 → 多様性解析 → 統計図 → レポート生成まで一気通貫 |
| **[qiime2-assistant](https://github.com/qiime-lab/qiime2-assistant)** | Ollama + Streamlit で動く QIIME 2 対話式 AI アシスタント。マニュアルをナレッジベースとしてチャット形式で質問に回答 |
| **[qiime2-manual](https://github.com/qiime-lab/qiime2-manual)** | 16S rRNA 解析の包括的日本語マニュアル。コピー＆ペーストで実行可能、内部標準による絶対定量ワークフローを含む |

## Tech Stack

`QIIME 2` · `DADA2` · `SILVA 138` · `Ollama` · `Streamlit` · `Python` · `pandas` · `scikit-learn` · `matplotlib`
